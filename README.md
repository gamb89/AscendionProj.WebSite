
# ASCENDION - TECHNICAL TEST 

This react application connects to AscendionProj.API to list the "best stories". It's part of the Asncedion hiring process

### Built With 
* React
* Vite

### Project Structure
src/
 ├── assets 
 
 ├── components → reusable UI components 
 
 ├── services -> service that consumes API
 
 ├── types -> interface
 
 └── App.tsx

### Installation

1. Clone the repository
 git clone https://github.com/username/project-name.git

2. Install dependencies

  npm install
  
  or
  
  yarn install

3. Run the development server

  npm run dev
  
  or
  
  npm start

### Additional Info
 * This application consumes the AscendionProj backend (API). Make sure that VITE_API_BASE_URL in ".env.example is correctly configured
 * I applied the Santander color palette (the challenge file mentions "Santander", so, I tried to match the design as closely as possible)
